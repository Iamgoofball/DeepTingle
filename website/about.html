Deep Tingle

Deep Tingle is a reasearch project. Our goal is to use machine learning to develop an assistant tool that can help someone to write like an specific target: in this case, Chuck Tingle. This system was developed with GloVe, Keras and Keras-JS.

Deep Tingle was developed by Ahmed Khalifa, Gabriella A. B. Barros and Julian Togelius.
